--- name: plsql-debugging-information-now-part-of layout: post title: PL/SQL debugging information now a part of Oracle Enhanced Adapter v1.2.2 time: 2009-10-02 15:39:00 -04:00 --- Raimonds Simanovskis has just published a version 1.2.2 of the Oracle Enhanced Adapter that includes the ability to capture dbms_output debug statements from your pl/sql code in the Rails log file. This is a bit of code that I wrote and blogged about a few months ago so not only do I think its useful but am very excited to have contributed to something many others use.
